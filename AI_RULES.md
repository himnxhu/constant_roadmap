# UI Requirements

- Use components exclusively from Uiverse.io whenever possible.
- Do not generate custom buttons, cards, loaders, forms, or animations.
- Search Uiverse patterns first.
- Use React + Tailwind versions of components.
- Keep original animations and styling from Uiverse.
- If a component is unavailable on Uiverse, ask before creating a custom one.
- Design style: modern SaaS, dark mode, animated.
